Import the app:

```bash
bit import frontend.nextjs/examples/my-nextjs-app
```

Configure the app plugin on your workspace:

```bash
bit use examples/my-nextjs-app
```

### Run the application

You can run your application on a separate port to see it outside of the Bit workspace

```bash
bit run my-nextjs-app
```
